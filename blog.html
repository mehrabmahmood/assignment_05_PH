Important FAQ

The Document Object Model (DOM) is a programming interface that represents a web page's structure as a tree, allowing dynamic manipulation of its content, structure, and style.

To select an element from the DOM, you can use:
1. document.querySelector('selector') for selecting the first matching element.
2. document.getElementById('id') for selecting an element by its unique ID.

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to handle events on its child elements using event bubbling. It improves performance and simplifies code by reducing the number of event listeners.

You can manipulate an element's attributes using setAttribute() or by directly modifying properties (e.g., element.id = "newId"). For styles, use element.style to modify CSS properties (e.g., element.style.color = "blue").
